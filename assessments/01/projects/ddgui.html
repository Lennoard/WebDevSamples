DD GUI

Uma interface gráfica para o bom e velho comando dd em um ambiente Android com root

DD GUI é um software livre e de código aberto (FOSS) cujo objetivo é facilitar a execução de comandos dd fornecendo uma interface gráfica.

Para resumir rapidamente o que pode ser feito com este comando, aqui está uma lista:

  • Faça backup da imagem atual do sistema/kernel/recuperação/[...] para qualquer lugar
  • Instale diretamente uma nova recuperação/kernel (se a partição de recuperação for conhecida e for um arquivo .img)
  • Copie/substitua arquivos como root
