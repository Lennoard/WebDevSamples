SysctlGUI

Um aplicativo como interface gráfica para editar variáveis do Kernel Android usando o comando sysctl.

Funcionalidades

  • Navegar no sistema de arquivos em busca de parâmetros específicos do kernel
  • Selecionar parâmetros de uma lista pesquisável
  • Informações sobre parâmetros conhecidos
  • Carregar parâmetros a partir de um arquivo de configuração
  • Reaplicar parâmetros na inicialização
  • Marcar os parâmetros como favoritos para facilitar o acesso

Tecnologias

  • MVVM para parâmetros de usuário
  • Jetpack Data Binding
  • Jetpack View Binding
  • Carregar parâmetros a partir de um arquivo de configuração
  • Kotlin Corrotinas com reconhecimento de ciclo de vida
  • Injeção de dependência com Koin
