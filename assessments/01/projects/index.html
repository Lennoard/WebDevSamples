SysctlGUI

Um aplicativo de código aberto cujo objetivo principal é fornecer uma maneira gráfica de editar os parâmetros do kernel. Esses parâmetros são aqueles listados em uma pasta especial do sistema e são editados usando o comando sysctl.

HEBF Optimizer

Um app que visa melhorar o desempenho do seu dispositivo ao se concentrar na economia de bateria, ajustando os parâmetros do sistema Android e/ou do kernel usando privilégios de superusuário adquiridos.

DD GUI

Uma interface gráfica para o bom e velho comando dd em um ambiente Android com root. DD GUI é um software livre e de código aberto (FOSS) cujo objetivo é facilitar a execução de comandos dd fornecendo uma interface gráfica.
